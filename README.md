# oktopios-installer

Installers and setup assets for Oktopios across Windows, Linux, macOS, Android Termux, and iOS iSH.

## Contents

- `windows/` - Windows setup files and executable installer.
- `linux/` - Linux installer assets.
- `mac/` - macOS installer assets.
- `android-temux/` - Android Termux installer assets.
- `ios-ish/` - iOS iSH installer assets.

Main project: https://github.com/ALISOULEMOUANWIYA/oktopios
