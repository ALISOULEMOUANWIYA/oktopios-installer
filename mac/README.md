# Oktopios Installer - macOS

## Native installer script

Use the installer in:

```text
mac/white-setup/install-oktopios.command
```

Double-click it in Finder, or run:

```bash
cd mac/white-setup
chmod +x install-oktopios.command
./install-oktopios.command
```

## Command line

The command-line installation is handled by PyPI:

```bash
python3 -m pip install --user oktopios
okp --version
```
